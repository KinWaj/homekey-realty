# 🗝️ HomeKey Realty

A responsive real-estate landing page built with **Next.js (App Router)** and **Tailwind CSS**, translated from the HomeKey Realty Figma design.

##✨ Tech stack
- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3

##🚀 Getting started
```bash
npm install
npm run dev      # http://localhost:3000
```

##🔧 Build
```bash
npm run build
npm run start
```
